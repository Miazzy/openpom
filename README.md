# OpenPom单机版舆情监控软件源码

系统使用struts+spring+hibernate框架开发，相关技术包括：redis,lucene,jms,nutch,rpc等。<br/><br/>
支持搜索引擎搜索结果、网站、论坛、博客、微博、微信公众号文章采集，支持电子邮箱、QQ聊天、微信记录监控。<br/><br/>
搜索结果自动规则化解析，提取文章标题、内容、发布时间、作者，自动生成摘要、关键词信息。自动聚类，判断文章正负面属性。<br/><br/>
